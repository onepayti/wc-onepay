O Plugin oficial OnePay para WooCommerce.

== Description  ==

Com o plugin oficial da OnePay para WooCommerce, sua loja virtual Woocommerce pode processar todas as suas vendas, permitindo que seus clientes realizem as compras por boleto bancário e em diferentes bandeiras de cartões de crédito.
A instalação é bem simples e não necessita que você tenha conhecimentos técnicos: 
Seguindo o passo a passo de instalação pelo nosso site de suporte e documentações você já começa a vender.

*SIMPLES E OBJETIVO*

O sistema possui um desenvolvimento preparado para atender a todos os segmentos de mercado!
A solução OnePay para WooCommerce possui um sistema para prevenção de fraudes nível 5, com todos os parâmetros de segurança necessários. Sendo assim, você não precisa se preocupar com os pagamentos, isso é por conta da OnePay. Tenha o foco em vender e fazer seus clientes felizes!

== Instalação ==

1. Acesse o painel de administrador do WordPress.
2. Na barra lateral esquerda, vá para: Plugins > Adicionar novo.
3. No campo de buscas entre com "OnePay para WooCommerce". 
4. Selecione nosso plugin na lista e clique em "Instalar agora".
5. Por fim, clique no botão "Ativar" após a instalação.

= Instalação Manual =

O método de instalação manual envolve o download do nosso plugin e o seu upload para o seu servidor web através do seu aplicativo FTP favorito. O codex do WordPress contém [instruções sobre como fazer isso aqui](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Requerimentos ==

- Ao utilizar o plugin OnePay para WooCommerce, você está de acordo com nossa [Política de Privacidade](https://www.1pay.com.br/politica-de-privacidade/) e nossos [Termos de Uso](https://www.1pay.com.br/termos-de-uso/)
- Uma conta OnePay. [Clique aqui](https://dash.1pay.com.br/pre-cadastro-ec) para solicitar.
- Credenciais de acesso (token) para API. [Entre em contato com a equipe OnePay para obtê-las.](https://www.1pay.com.br/)
- Um certificao SSL (Recomendado)
- Plugin "WooCommerce" ativado e instalado
- Versão mínima do PHP  7.0

== Frequently Asked Questions ==

= Qual país a OnePay atende? =
Atualmente a OnePay recebe pagamentos somente e exclusivamente no Brasil.

= Onde posso obter suporte ou conversar com outros usuários? =
Se você tiver problemas, pode pedir ajuda no fórum de plugins ou entrar em contato com: suporte.ti@1pay.com.br

= Tenho que fazer backup antes? =
Não necessariamente! Como qualquer desenvolvedor ou usuário do Wordpress, criamos a mania de segurança e antes de instalar qualquer plugin faça o backup do banco de dados e arquivos.

= Onde posso acompanhar as vendas realizadas? =
As vendas poderão ser acompanhadas em nosso [Dashboard](https://dash.1pay.com.br/) e também em nossa [API](https://docs.1pay.com.br/).

= O pedido foi pago e ficou com o status de "processando" e não como concluído, isto está certo? =
Sim, está certo e o plugin está funcionando como deveria.

= Problemas com a integração? =
Ative a opção **Log de depuração** e tente realizar o pagamento novamente.
Feito isso, copie o conteúdo do log e salve usando o [pastebin.com](https://pastebin.com/) ou o [gist.github.com](gist.github.com), depois basta abrir um tópico de suporte [aqui](https://wordpress.org/support/plugin/onepay-for-woocommerce/).
