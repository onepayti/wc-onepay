O Plugin oficial OnePay para WooCommerce.

== Description ==

= Em Português Brasil =
O plugin OnePay para WooCommerce permite que lojistas aceitem pagamentos por boleto e cartão de crédito usando a OnePay no WooCommerce.

= in English =
The OnePay plugin for WooCommerce allows merchants to accept payment by boleto and credit card using OnePay on WooCommerce.

== Installation ==

1. Acesse o painel de administrador do WordPress.
2. Na barra lateral direita, vá para: Plugins > Adicionar novo.
3. No campo de buscas entre com "OnePay para WooCommerce". 
4. Selecione nosso plugin na lista e clique em "Instalar agora".
5. Por fim, clique no botão "Ativar" após a instalação.

= Manual Installation =

O método de instalação manual envolve o download do nosso plugin e o seu upload para o seu servidor web através do seu aplicativo FTP favorito. O codex do WordPress contém [instruções sobre como fazer isso aqui](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Requirements ==

- Uma conta OnePay
- Credenciais de acesso (token) para API. [Entre em contato com a equipe OnePay para obtê-las.](https://www.1pay.com.br/)
- Um certificao SSL (Recomendado)
- Plugin "WooCommerce" ativado e instalado.
- Versão mínima do PHP  7.0

== Frequently Asked Questions ==

= Qual país a OnePay atende? =
Atualmente a OnePay recebe pagamento somente e exclusivamente no Brasil.

= Onde posso obter suporte ou conversar com outros usuários? =
Se você tiver problemas, pode pedir ajuda no fórum de plugins ou entrar em contato com: suporte.ti@1pay.com.br

= Tenho que fazer backup antes? =
Não necessariamente! Como qualquer desenvolvedor ou usuário do wordpress, criamos a mania de segurança e antes de instalar qualquer plugin faça o backup do banco de dados e arquivos.
